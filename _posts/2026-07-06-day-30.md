what_i_learned: |
 Today I learned about the coding aspect of deep learning by implementing different types of recurrent neural networks, including SimpleRNN, Long Short-Term Memory (LSTM), Gated Recurrent Unit (GRU), and Bidirectional recurrent networks. Writing the code helped me understand how these models process sequential data and how they differ in handling information over time. I learned that a SimpleRNN is the most basic recurrent neural network, but it struggles to remember information from long sequences because of the vanishing gradient problem. LSTM networks solve this issue by using memory cells and gates that determine what information to keep, update, or forget. I also learned that GRU models provide a simpler alternative to LSTMs by using fewer gates while still maintaining strong performance on many sequence prediction tasks.

  In addition, I learned how Bidirectional recurrent neural networks improve predictions by processing sequential data in both forward and backward directions. Coding these models gave me a better understanding of how to define network layers, select activation functions, and train deep learning models using sequential datasets. I now understand why LSTM and GRU networks are commonly used for time-series forecasting, natural language processing, and other applications where previous context is important. Comparing the code for each architecture helped me recognize the trade-offs between computational complexity, training speed, and prediction accuracy. Overall, this lesson strengthened both my programming skills and my understanding of how different deep learning architectures are designed to capture patterns in sequential data.
