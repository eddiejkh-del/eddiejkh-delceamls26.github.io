what_i_learned: |
 Today I learned more about data analysis and the tools that are commonly used to explore and visualize datasets. One of the libraries I studied was Seaborn (sns), which is built on top of Matplotlib and provides an easier way to create attractive statistical graphics. I practiced using Seaborn to generate plots such as scatter plots, heatmaps, and box plots that help identify trends and relationships within data. These visualizations make it easier to understand large datasets and communicate findings effectively. I also learned that data visualization is an important step before building machine learning models because it helps reveal patterns, outliers, and potential issues with the data. By using these tools, I gained a better understanding of how data can be analyzed and presented in a meaningful way.

  In addition, I learned more about classification models and how they are used to predict categories or classes based on input data. I explored how models make predictions and how their performance can be evaluated using a confusion matrix. A confusion matrix breaks down predictions into true positives, true negatives, false positives, and false negatives, making it easier to identify where a model performs well or makes mistakes. Understanding these metrics helps determine the accuracy and reliability of a classification model beyond a single accuracy score. Learning about Seaborn, classification models, and confusion matrices has given me a stronger foundation in data analysis and machine learning and has shown me how visualization and evaluation work together to improve predictive models.
